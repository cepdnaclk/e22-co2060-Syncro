\# Phase 1 Overview (Syncro)



Goal: Build the MVP workflow:

Client posts RFP → sellers bid → client awards bid.



Core features:

1\. Sign up / login

2\. Role selection (Client / Seller)

3\. Post RFP with niche/category

4\. Sellers view relevant RFPs

5\. Sellers submit bids

6\. Client selects winning bid

7\. Basic statuses (Open → Awarded → Closed)
