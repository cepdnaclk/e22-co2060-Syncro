# SYNCRO-2YP
Universal Reverse-Auction Marketplace 

Syncro is a universal reverse-auction marketplace where clients post requests
and verified sellers compete by submitting bids.

## Project Scope
- Dual-role users (Client / Seller)
- Request for Proposal (RFP) posting
- Niche-based seller notifications
- Competitive bidding system

## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js (Express) or FastAPI
- Database: PostgreSQL
- Real-time: WebSockets
